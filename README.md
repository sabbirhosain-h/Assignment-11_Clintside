## 📖 Project Overview
BookCourier is a modern library delivery management system where users can request book pickup or delivery from nearby libraries. It helps students, researchers, and readers borrow and return books without physically visiting the library.

## 🎯 Purpose
- Simplify book borrowing & delivery
- Provide role-based dashboards (User, Librarian, Admin)
- Enable secure online book ordering and management

## ✨ Key Features
- Responsive modern UI (mobile, tablet & desktop)
- Firebase authentication (Email/Password + Social Login)
- Role-based dashboards
- Book ordering & tracking system
- Wishlist functionality
- Book review & rating system
- Search & sort books by name and price
- Light/Dark mode toggle
- Protected routes with JWT authentication
- Secure API integration

## 🧩 Pages & Layout
- Home (Banner slider, latest books, coverage map, animated sections)
- All Books
- Book Details (Order modal)
- Dashboard (User / Librarian / Admin)
- Authentication (Login & Register)
